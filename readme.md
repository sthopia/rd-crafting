## rd-crafting
crafting with succesrate which causes it to explosion you can edit it in config.lua.
![ShowCase](https://cdn.discordapp.com/attachments/806431193352503319/849208585485418506/config.PNG)

## Requirements
1. [mythic_progbar](https://github.com/HalCroves/mythic_progbar) 
2. [mythic_notify](https://github.com/JayMontana36/mythic_notify) 

## Installation
1. Put rd-crafting file into resources
2. Add start rd-craftingon your server.cfg
